NYT Bee Command Line
MS Windows Installation

There is a YouTube video that shows the installation process below.

Click on this link: nytbee.exe to download the command line program - to your Downloads folder.

Microsoft rightfully worries about people downloading and installing unknown software. The video above shows how you can affirm that the nytbee.exe program is okay to install.

Move the nytbee.exe file from Downloads to the Desktop.

You can now simply double click the desktop nytbee icon to start the program and you will be solving today's puzzle!
