NYT Bee Command Line
Apple Mac Installation

There is a YouTube video that shows the installation process below.

Click on this link: nytbee.pl to download the command line program to your Downloads folder.

The program is written in the Perl language. Perl is already installed on Apple machines.

The program will run inside the Terminal application and it needs the following configuration steps:

  1. In the Finder click on Applications.
  2. In the Utilities folder find Terminal. Drag and drop it to a convenient place in the Dock.
  3. Click on Terminal in the Dock to open it.
  4. Alternatively, you could create a Terminal icon on the Desktop and double click it to open it.
  5. From the top-level menu of Terminal choose Terminal > Preferences.
  6. On the Text tab choose Change to change the font size to 24.
  7. On the Shell tab you will check the Run Command check box and enter this in the text field: 
      perl ~/Downloads/nytbee.pl
    The funny character ~ above is a tilde. It is found at the upper left of your keyboard.
  8. Run inside shell should be checked.
  9. Close the Preferences dialog.
  10. Quit Terminal

Now you are all set. Each day you simply click on Terminal in the Dock (or the Desktop) and you will be playing today's NYT Bee puzzle. It looks like this: 

    12/14/21
Enter h for help.

   D   G
 I   H   N
   O   T

>
When you finish and quit the NYT Bee program you can also quit Terminal.
